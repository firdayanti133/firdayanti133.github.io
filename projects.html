Ayodonor This blood donation application is a web-based blood donation management system designed to facilitate interaction between users and providers. The main purpose of this application is to help users gain access to the blood they need, while providers can manage blood stocks efficiently.

TryItOn

TryItOn is a fashion e-commerce application with excellent features that are different from other platforms. These features are the AR (Augmented Reality) Camera, and 3D Modeling.

The AR camera is designed to make it easier for potential buyers to try on clothes virtually using a camera. Then, the author adds a 3D Modeling feature that functions to display product details clearly and interactively, so that you can see every corner of the product more easily.
